Blender Projects

Head Sculpt

3D Character Creation has been one of my biggest goals to reach in Blender so when I felt that I had gotten a decent grasp of the basics of modeling and sculpting in Blender, I decided to take a shot at creating a head based on references that I drew to see if I could do it.

Chum Bucket

I decided to do this interpretation of the Chum Bucket from SpongeBob because one of my friends did it and I wanted to see if I could do it better. It is based on a reference picture that the friend found online of a derelict Chum Bucket in the future.

Red Bull Can

I decided to put this Red Bull can close to the top mainly because I love Red Bull. I made this can using a tutorial as this whas when I decided to start taking Blender a bit more seriously and wanted to start modeling something I really liked.

Spiky Dude

I made this character for my Main Evidence 2 game for Game Production 2 called Try or Die. It is based on a 2D top down drawing of an enemy I created for my Main Evidence 1 game for Game Production 2 called Get Those Alien Things. This enemy chases you down and clings onto you while doing damage.

Aimer

This enemy character was also made for the game Try or Die. It is a character that shoots at you from long range with energy blasts.

Try or Die Gun

This is the gun that the player uses in the game Try or Die. It shoots red energy blasts.
